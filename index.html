Introducción

Java es un lenguaje de programación orientado a objetos, multiplataforma y muy popular para desarrollar aplicaciones de todo tipo, desde aplicaciones móviles hasta sistemas empresariales.

Fue creado por Sun Microsystems en 1995 y es conocido por su portabilidad gracias a la máquina virtual Java (JVM).

1. Tipos de Variables en Java

Java es un lenguaje fuertemente tipado, lo que significa que cada variable debe declararse con un tipo específico. Los tipos de variables se dividen en:

Tipos Primitivos

  • byte: 8 bits, rango de -128 a 127.
  • short: 16 bits, rango de -32,768 a 32,767.
  • int: 32 bits, rango de -2,147,483,648 a 2,147,483,647.
  • long: 64 bits, rango muy grande.
  • float: 32 bits, números decimales con precisión simple.
  • double: 64 bits, números decimales con precisión doble.
  • char: 16 bits, representa un solo carácter Unicode.
  • boolean: representa valores true o false.

Tipos de Referencia

  • Clases: objetos creados a partir de clases.
  • Arrays: colecciones de elementos del mismo tipo.
  • Interfaces: tipos abstractos que definen métodos.

2. Bucles en Java

Los bucles permiten repetir bloques de código mientras se cumpla una condición.

Bucle for

for(int i = 0; i < 10; i++) {
System.out.println(i);

}

Bucle while

int i = 0; while(i < 10) { System.out.println(i); i++; }

Bucle do-while

int i = 0; do { System.out.println(i); i++; } while(i < 10);

3. Puertas Lógicas y Operadores Lógicos

Java soporta operadores lógicos para controlar el flujo de programas:

  • && (AND): true si ambos operandos son true.
  • || (OR): true si al menos un operando es true.
  • ! (NOT): invierte el valor lógico.

Ejemplo:

boolean a = true;

boolean b = false; if(a && !b) { System.out.println("Condición verdadera"); }

4. Ejecuciones y Control de Flujo

Java permite controlar el flujo de ejecución con estructuras como:

  • if / else: Condicionales.
  • switch: Selección múltiple.
  • break: Salir de bucles o switch.
  • continue: Saltar a la siguiente iteración del bucle.

5. Métodos en Java

Los métodos son bloques de código que realizan tareas específicas y pueden recibir parámetros y devolver valores.

Declaración de un método

public int sumar(int a, int b) {
return a + b;

}

Llamada a un método

int resultado = sumar(5, 3); System.out.println(resultado); // 8

Métodos estáticos y de instancia

  • Estáticos: pertenecen a la clase y se llaman sin crear un objeto.
  • De instancia: pertenecen a un objeto y requieren una instancia para ser llamados.

6. Clases y Objetos

Java es un lenguaje orientado a objetos. Una clase es un molde para crear objetos.

Definición de una clase

public class Persona {
String nombre;
int edad;

public Persona(String nombre, int edad) {
    this.nombre = nombre;
    this.edad = edad;
}

public void saludar() {
    System.out.println("Hola, soy " + nombre);
}

}

Crear un objeto

Persona p = new Persona("Ana", 30); p.saludar();

7. Manejo de Excepciones

Permite manejar errores en tiempo de ejecución para evitar que el programa termine abruptamente.

Try-catch

try {
int division = 10 / 0;

} catch(ArithmeticException e) { System.out.println("Error: División por cero"); }

Finally

Bloque que se ejecuta siempre, haya o no excepción.

8. Arrays

Estructuras para almacenar múltiples valores del mismo tipo.

int[] numeros = {1, 2, 3, 4, 5};

System.out.println(numeros[0]); // Imprime 1

9. Colecciones

Java ofrece colecciones como ArrayList, HashMap, etc., para manejar grupos de objetos con más flexibilidad.

10. Modificadores de Acceso

  • public: Accesible desde cualquier clase.
  • private: Solo accesible dentro de la misma clase.
  • protected: Accesible dentro del paquete y subclases.

11. Herencia

Permite que una clase herede propiedades y métodos de otra.

public class Animal {
public void comer() {
    System.out.println("Comiendo...");
}

}

public class Perro extends Animal { public void ladrar() { System.out.println("Guau Guau"); } }

12. Interfaces

Definen métodos que las clases deben implementar.

public interface Volador {
void volar();

}

public class Pajaro implements Volador { public void volar() { System.out.println("Volando"); } }

13. Programación Genérica

Permite crear clases y métodos que operan con tipos genéricos.

public class Caja<T> {
private T contenido;

public void setContenido(T contenido) {
    this.contenido = contenido;
}

public T getContenido() {
    return contenido;
}

}

14. Concurrencia y Hilos

Java permite ejecutar múltiples hilos para tareas concurrentes.

public class MiHilo extends Thread {
public void run() {
    System.out.println("Hilo en ejecución");
}

}

MiHilo hilo = new MiHilo(); hilo.start();

15. Entrada y Salida (I/O)

Java ofrece clases para manejar entrada y salida de datos, como Scanner para entrada por consola.

import java.util.Scanner;

Scanner sc = new Scanner(System.in); System.out.print("Introduce tu nombre: "); String nombre = sc.nextLine(); System.out.println("Hola " + nombre);
